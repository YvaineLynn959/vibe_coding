# Project Name
vibe_coding

## Project Description
Personal portfolio website for Zhang Tianyu - Brand Designer & Art Director

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
- Google Fonts (Inter, JetBrains Mono, Playfair Display)

## Features
- Responsive design
- Animated hero section
- Custom cursor effects
- Scroll animations
- Mobile navigation

- Accent (Purple): #8B5CF6
- Gold: #C9A962

## Typography
- Display: Playfair Display
- Body: Inter
